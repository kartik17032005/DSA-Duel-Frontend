# DSA Duel Android 📱⚔️

DSA Duel Android is a modern native Android application built with Kotlin and Jetpack Compose that transforms Data Structures & Algorithms practice into a competitive and engaging experience. Users can track progress, maintain streaks, climb rankings, and unlock new topics as they improve their problem-solving skills.

## ✨ Features

### User Experience

* Modern Jetpack Compose UI
* Material Design 3 Components
* Smooth Navigation & Animations
* Responsive User Interface

### Competitive Learning

* Elo-Based Ranking System
* Global Leaderboards
* Performance Tracking
* Competitive Coding Experience

### Gamification

* Daily Streak Tracking
* Topic Progression System
* Unlockable Learning Paths
* Achievement-Based Motivation

### Learning Journey

* Structured DSA Roadmap
* Topic-wise Progress Monitoring
* Personalized User Dashboard
* Detailed Performance Insights

## 🚀 Getting Started

### Prerequisites

* Android Studio
* JDK 17+
* Android SDK
* Running DSA Duel Backend

### Clone Repository

```bash
git clone https://github.com/kartik17032005/dsa-duel-android.git
```

### Configure Backend URL

Update the backend API URL in your configuration files before running the application.

### Run Application

Open the project in Android Studio and run the application on an emulator or physical device.
